In linear algebra, an idempotent matrix is a [[id:7a43b0c7-b933-4e37-81b8-e5ecf9a83956][Matrix]] which, when multiplied by itself, yields itself. That is, the matrix \(A\) is idempotent if and only if \(A^2 = A\).
