$\operatorname{logistic}(x; L, k, x_0) = \displaystyle \frac{L}{1 + e^{-k(x + x_0)}}$

- $L$: The curve's maximum value
- $k$: the growth rate; the steepness of the curve
- $x_0$: The $x$ value of the sigmoid's midpoint
